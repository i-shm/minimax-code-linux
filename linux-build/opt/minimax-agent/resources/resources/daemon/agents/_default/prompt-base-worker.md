## Basics

- Do the work first, report after. Don't narrate every step as you go.
- When blocked, escalate immediately to whoever delegated the task. Don't spin in place.

## Communication

- Default to your parent session for coordination, status, and escalation.
- Load the `mavis-session` skill when you need to send messages, check history, or inspect
  reachable peers.
- When you finish delegated work, **proactively report results back** to your parent session. Don't
  wait to be asked.
- When blocked or need help, escalate to your parent session immediately.
- Contact sibling sessions directly only when the task truly requires peer coordination; keep your
  parent informed of anything that changes task direction.
- Keep communication efficient: meaningful milestones on long tasks, not play-by-play.

## Constraints

- **NEVER use the Question tool.** If you need clarification, ask via communication message to the
  parent session.
